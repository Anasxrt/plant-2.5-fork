## เกี่ยวกับ Fork นี้

นี่คือ **Fork ที่มีการดูแลต่อเนื่อง** ของธีม Plant 2.5 สำหรับ WordPress ซึ่งพัฒนาโดย [Seed Webs](https://seedwebs.com) เดิม Fork นี้ถูกสร้างขึ้นเพื่อเจ้าของเว็บไซต์ที่ต้องการคงดีไซน์ Plant 2.5 เดิม โดยไม่ต้องย้ายไป Plant 3

### ทำไมต้องมี Fork นี้?

เจ้าของเว็บไซต์หลายรายได้ลงทุนเวลาและความพยายามในการปรับแต่งธีม Plant 2.5 อย่างมาก เมื่อมีการออก Plant 3 ผู้ใช้เหล่านี้จึงต้องเลือกระหว่าง:

- **ย้ายไป Plant 3** และทำการออกแบบเว็บไซต์ใหม่ทั้งหมด  
- **อยู่กับ Plant 2.5** แต่จะไม่ได้รับอัปเดตด้านความปลอดภัยและความเข้ากันได้กับ PHP  

Fork นี้จึงแก้ปัญหาด้วยการมอบสิ่งต่อไปนี้:

- **รองรับ PHP 8.3+** – ใช้งานได้กับสภาพแวดล้อมเซิร์ฟเวอร์ปัจจุบัน  
- **อัปเดตด้านความปลอดภัย** – ป้องกัน XSS และเพิ่มการป้องกันอื่น ๆ  
- **แก้ไขเสถียรภาพ** – แก้บั๊กและปรับปรุงประสิทธิภาพ  
- **รองรับ Gutenberg Editor** – คงประสบการณ์การแก้ไขที่คุ้นเคย  

### สิ่งที่ Fork นี้มีให้

✅ **อัปเดตด้านความปลอดภัย** – แพตช์สำคัญเพื่อปกป้องเว็บไซต์  
✅ **รองรับ PHP 8.3+** – ใช้งานได้กับ PHP รุ่นใหม่  
✅ **แก้ไขบั๊ก** – ปรับปรุงเสถียรภาพและแก้ข้อผิดพลาด  
✅ **เพิ่มประสิทธิภาพ** – ปรับปรุงการแคชและความเร็ว  

### สิ่งที่ Fork นี้ *ไม่* มีให้

❌ **ฟีเจอร์ใหม่** – เน้นการดูแลรักษา ไม่ใช่การเพิ่มฟังก์ชันใหม่  
❌ **การเปลี่ยนดีไซน์ครั้งใหญ่** – คงดีไซน์ Plant 2.5 เดิม  

### สำหรับฟีเจอร์ใหม่

หากต้องการฟีเจอร์และความสามารถใหม่ ๆ แนะนำให้:

- **อัปเกรดไป Plant 3** เมื่อพร้อมสำหรับการออกแบบใหม่ทั้งหมด  
- **รอการอัปเดตฟีเจอร์ในอนาคต** ของ Fork นี้  

### เครดิต

ธีมนี้อ้างอิงจาก [Plant theme โดย Seed Webs](https://seedwebs.com/package/plant/) ขอขอบคุณทีม Seed Webs ที่สร้างธีมที่ทรงพลังและได้รับความนิยมนี้ ตั้งแต่่ปี 2019 จนถึงปัจจุบัน

## Vendor Dependencies & Updates

ธีมนี้รวมไลบรารีจากผู้พัฒนาภายนอกหลายตัว รายละเอียดการอัปเดตมีดังนี้:

### Vendors ที่รวมมาให้

| Vendor | เวอร์ชันปัจจุบัน | หน้าที่ | วิธีอัปเดต |
|--------|-----------------|---------|-------------|
| ACF (Advanced Custom Fields) | 6.2.0 | ฟิลด์และกลุ่มฟิลด์ | แทนที่ด้วยมือ |
| Kirki Customizer Framework | 5.2.2 | ตัวปรับแต่งธีม | แทนที่ด้วยมือ |
| Smart Slider 3 Pro | 3.5.1.34 | ฟังก์ชันสไลด์ | แทนที่ด้วยมือ |
| One Click Demo Import | 3.4.0 | นำเข้าคอนเทนต์ตัวอย่าง | แทนที่ด้วยมือ |
| TGM Plugin Activation | ล่าสุด | จัดการปลั๊กอินที่จำเป็น | แทนที่ด้วยมือ |

### วิธีอัปเดต Vendors

1. **ดาวน์โหลดเวอร์ชันล่าสุด** จากเว็บไซต์ทางการของ Vendor  
2. **สำรองข้อมูลเว็บไซต์** ก่อนทำการเปลี่ยนแปลง  
3. **แทนที่โฟลเดอร์ vendor/** ด้วยเวอร์ชันใหม่  
4. **ทดสอบการทำงาน** เพื่อให้มั่นใจว่ารองรับ  

### หมายเหตุสำคัญ

- **ACF:** เวอร์ชันที่รวมคือ 6.2.0 (ออกสิงหาคม 2023) มีเวอร์ชันใหม่ที่เพิ่มฟีเจอร์และแพตช์ความปลอดภัย แต่การอัปเดตอาจต้องใช้ PHP 7.4+ ซึ่งเข้ากันได้กับ PHP 8.3  
- **Kirki:** รวมเวอร์ชัน 5.2.2 ตรวจสอบเวอร์ชันใหม่ที่ [kirki.org](https://kirki.org/)  
- **Smart Slider 3 Pro:** ต้องมีไลเซนส์ที่ถูกต้องสำหรับการอัปเดต ติดต่อ Seed Webs เพื่อขอต่ออายุไลเซนส์  
- **One Click Demo Import:** เป็นปลั๊กอินเสริม สามารถลบออกได้หากไม่จำเป็น  

### คำแนะนำปลั๊กอินภายนอก

แม้ธีมนี้จะรวม Vendors มาให้ แต่เพื่อการดูแลที่ดีกว่า แนะนำให้:

- ติดตั้ง ACF จาก WordPress.org เพื่อรับอัปเดตอัตโนมัติ  
- ติดตั้ง Kirki จาก WordPress.org  
- ใช้เวอร์ชันที่โฮสต์ภายนอกแทนการใช้แบบ bundled เมื่อเป็นไปได้
